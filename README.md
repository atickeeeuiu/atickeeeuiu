## 🧠 Research Experience

During my PhD research at **Nanjing University of Science and Technology**, I developed graph-based and deep learning frameworks for EEG emotion recognition, including:

* **H-GAT**: Hybrid Graph Attention Network for multidimensional EEG feature learning
* **Multilayer-GTCN**: Graph Transformer-based model using physical and functional EEG connectivity
* **Domain-Adaptive EEG Modeling**: Cross-dataset EEG emotion recognition using domain adaptation
* **SA-DEEF**: Subject-adaptive differential-entropy ensemble framework with spectral-spatial interpretability

My work involves EEG preprocessing, feature extraction, graph construction, model training, performance evaluation, and interpretability analysis using attention visualization, t-SNE, confusion matrices, and node-importance analysis.
